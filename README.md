# bigdata-analytics-kimiafarma-x-rakamin-pbi
Repositori ini berisi hasil proyek dari Program Project-Based Virtual Internship yang diselenggarakan oleh Rakamin Academy bekerja sama dengan Kimia Farma. Proyek ini berfokus pada penerapan Big Data Analytics, mencakup proses data cleaning, EDA, serta pembuatan dashboard interaktif untuk menghasilkan insight bisnis yang bernilai.
